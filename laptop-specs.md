# Lenovo ThinkBook 13s G4 ARB

## Hardware Specifications

| Component | Details |
|-----------|---------|
| **CPU** | AMD Ryzen 7 6800U (8 cores, 16 threads, up to 4.77 GHz) |
| **GPU** | AMD Radeon 680M (RDNA 2 integrated) |
| **RAM** | 12 GB LPDDR5 (soldered) |
| **Storage** | 512 GB NVMe SSD (Micron MTFDKCD512TFK) |
| **Display** | 13.3" 1920x1200 (16:10), 100% sRGB |
| **Battery** | 56Wh Li-Polymer, Rapid Charge |
| **WiFi** | Wi-Fi 6E (802.11ax) |

## Cache

- L1d: 256 KiB (8 instances)
- L1i: 256 KiB (8 instances)
- L2: 4 MiB (8 instances)
- L3: 16 MiB (shared)

## External Display Support

- HDMI-A-1: Up to 3840x2160@60Hz
